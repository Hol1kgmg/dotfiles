# Development Plan

## 作業フロー

各タスクを進める際の標準的なワークフロー：

1. **プランドキュメント作成** - タスク専用の`.md`ファイルを作成（例: `vscode-keybindings-plan.md`）
2. **プランに記入** - 実装内容、要件、設計をドキュメントに記入
3. **実装** - プランに基づいて実装を進める
4. **完了** - プランドキュメントを削除
5. **DEVELOPMENT_PLANを更新** - 該当タスクを完了としてマーク
